# Documentation

The connect-kotlin-server documentation site, built with [Astro](https://astro.build) and
[Starlight](https://starlight.astro.build). Content lives in `src/content/docs/`.

```bash
pnpm install
pnpm dev       # local dev server
pnpm build     # static build into dist/
pnpm preview   # preview the production build
```

It is configured as a GitHub Pages project site under `/connect-kotlin-server` — see
`astro.config.mjs`.
